## 详细解释

### Apartment-公寓

| Item        | Value                                                                  |
| ----------- | ---------------------------------------------------------------------- |
| **单词**    | apartment                                                              |
| **中译**    | 公寓                                                                   |
| **音标**    | `[əpɑ́ːrtmənt]`                                                         |
| **Meaning** | An <i>apartment</i> is a set of rooms in a building where people live. |
| **Example** | She has a nice <b>apartment</b> in the city.                           |
